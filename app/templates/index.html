Sekilas

Selamat datang di SHOLAT-API !

API sederhana untuk mendapat jadwal sholat untuk kota-kota di Indonesia. Waktu yang tertera merupakan Waktu Indonesia Barat (WIB) atau GMT+7

Ini hanyalah API hasil scraping web https://www.jadwalsholat.org

Endpoint

  1. /jadwal

    2. Endpoint untuk mengambil jadwal sholat

    Metode: GET

    Paramenter:

    • id_kota

      • id kota, bisa dilihat menggunakan endpoint /kota.

    • mode

      • terdiri dari 2 mode, 'daily' & 'monthly.

    Contoh:

    /jadwal?id_kota=30&mode=daily

    Akan mengambil jadwal sholat hari ini untuk kota Batam

    Respon Hasil:

    {
"success": true,
"debug_message": "ok",
"result": {
    "kota": "Batam",
    "data": {
        "tanggal": "07",
        "imsyak": "04:37",
        "shubuh": "04:47",
        "terbit": "06:01",
        "dhuha": "06:25",
        "dzuhur": "12:08",
        "ashr": "15:18",
        "maghrib": "18:12",
        "isya": "19:21"
        }
    }
}
